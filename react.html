30‑Day React Roadmap — 30 minutes/day

A focused, daily plan to learn React efficiently. Small steps, big results.

Week 1 — Core JavaScript Foundations

React is JavaScript — start by tightening the fundamentals you'll use every day.

DayFocus
1Intro & setup: install Node & VS Code or open CodeSandbox
2ES6 basics: let / const, arrow functions
3Destructuring arrays & objects
4Spread & rest operator (...)
5Array methods: .map(), .filter(), .reduce()
6Work with objects & JSON
7Mini‑project: plain JS to‑do list

Week 2 — React Basics

Start building with components, JSX, props and events.

DayFocus
8Create app (Vite or CodeSandbox)
9JSX syntax & differences from HTML
10Components: build & reuse
11Props: pass data between components
12Lists & keys
13Events: onClick, onChange
14Mini‑project: User Card component

Week 3 — State, Hooks & Styling

Make your UI interactive and nicely styled.

DayFocus
15useState hook
16Conditional rendering
17useEffect hook
18Forms & input handling
19Styling: CSS modules or Tailwind
20Props vs state
21Mini‑project: Counter / Weather fetcher

Week 4 — Advanced Topics & Final Project

Bring everything together and ship a small product.

DayFocus
22Component tree & data flow
23React Router (navigation)
24API calls with fetch or axios
25useRef & custom hooks
26Context API for global state
27Deployment: Netlify / GitHub Pages / Vercel
28–30Final project: e.g., Dyno Tech landing page or portfolio

Daily routine suggestion: 5min review • 15min learning • 10min practice. Keep consistency — 30 minutes a day adds up fast.
